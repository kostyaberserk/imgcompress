# imgcompress

**imgcompress** is a universal image compression library designed to work
with **bytes**, not files.

Perfect for:
- Telegram bots
- Web backends (FastAPI / Django)
- Microservices
- Desktop applications
- In-memory processing

## Features

- IO-agnostic (bytes → bytes)
- JPEG / PNG / WEBP support
- Simple compression level (1–10)
- Zero file system dependency
- Clean and minimal API

## Installation

```bash
pip install imgcompress
```
